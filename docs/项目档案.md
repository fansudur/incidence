# 镜面反射装置 · 项目档案

> 一个持续 15 年以上的长期艺术装置项目的档案。
> 供协作者理解项目的思想脉络、几何结构与演化历程。
> 配套文档：《协作导则》（约束规则）、《系统架构与工作计划》（技术实现）。

---

## 一、项目本质（一句话）

一个**用光路反射递归生成多层世界**的几何装置：一个球体（意识/光/起点）穿过一个取景框，投射到 45° 倾斜的镜面上，反射生成层层展开的世界。装置探讨「光与意识对物理世界的投射」「沟通的隔阂」「社会阶级 / 个体命运的几何结构」「世界的生成与消解」。

**它首先是一个几何上严格成立的物理系统，其次才承载哲学含义。几何本身承载意义，不需要文字解说。**

---

## 二、核心几何结构（基础原型）

```
Σ  球体 —— 意识 / 光 / 视线 / 创世起点。睁开眼睛即世界存在。
   │       （叠加态：在被触发前，可能是 1 也可能是 0，存在亦不存在）
   ↓  穿过
F  取景框（矩形）—— 世界被允许存在的形式约束 / 视锥限定
   │       光锥发散投射
   ↓  击中
M₁ 第一面镜面（倾斜 45°）—— 创世镜面
   │       45° 使光路旋转正好 90°（装置的"宇宙常数"）
   ↓  反射生成
W₁ 红色层 —— 第一层活动域
   │       光路偏移一段距离后遇到
M₂ → W₂ 绿色层 —— 第二层活动域
M₃ → W₃ 蓝色层 —— 第三层活动域
```

### 必须严守的几何不变量

1. **镜面 45° → 光路旋转 90°**。这是核心。每经过一面镜面，"世界的法线方向"旋转 90°，每四层回到初始姿态。
2. **层间相似放大**：W₁ 最小（离镜面/视点最近），W₂、W₃ 依次放大。放大比 ≈ 距离比的平方。最外层镜面 M₃ 往往极大，越靠近 Σ 的镜面越小。
3. **红/绿/蓝 = 一个世界内部的三个层级，不是三个独立世界**。它们都由同一个 Σ + F 生成，在 Σ 视点处合成为一幅完整画面。这是绝不能搞错的核心设定。
4. **bug 区域**：每层镜面尺寸有限。光路落在镜面物理边界之外 = 丢失 = 概念上的"穿帮 / 坠入黑暗 / 混沌"。

### 两个关键的"诠释维度"（同时成立）

红/绿/蓝三层可以被解读为——

- **解读 A（共时 / 空间）**：社会的三个阶级层。红=底层（空间小、资源少、吵闹）；绿=中产（体面）；蓝=富人（空间大、资源多、高高在上）。跨层极难（阶级跃迁），向下坠落则可能（投资失败 → 跌回下层，或直接坠入黑暗/混沌而不经历底层）。
- **解读 B（历时 / 时间）**：一个人一生的因果路径。在绿色区域所处的位置，对应在红/蓝区域的位置。「从第一个选择开始，就已经有了自己的光的路径」。偏离这条光路 = "死"。

**两种解读同时成立 ⇒ 社会几何与个人几何同构**（"As above, so below"）。这是装置最深的一层：同一个几何法则既描述社会横截面，也描述个人纵截面。

### Σ 的本体论地位

装置设定：**Σ 是理想化的、全知的造物主视点**——在 Σ 位置可同时看清每一层的完整生活（"全知全能造物主、对神的期待"）。
- 技术上存在"焦距问题"（不同层距离不同，相机难以同时对焦各层），但这是**技术性考量、非本体设定**：可用漫游/镜头切换弥补；不要把 Σ 解读为"有限/受难的神"。

## 三、五个变体（演化谱系）

变体不是平行的独立设计，而是**同一个生成系统在不同深度的展开**。

### 变体一：基础原型（单世界 · 阶级/人生分层）
单 Σ，纯反射，红/绿/蓝三层纵向分层。已完成第一个 web 版本。

### 变体二：半透明镜面 · 正背双世界
M₁ 不是纯反射镜，而是**半透明介质**。创世光击中 M₁ 时分裂为：反射分量（生成正面世界）+ 透射分量（在 M₁ 背面生成另一个世界）。
- 两个世界共享同一创世瞬间、同一 Σ、同一 F、同一 M₁，但**不对等**：因几何不对称（两侧到 M₁ 距离不同、视张角不同），两个世界尺寸不同、接收的能量/信息不同。
- 同一块镜面，对两侧居民呈现为不同大小的「天空」。
- 两个世界通常互相看不见（演示里把背面"挖空"只是为了让观众看清，实际是不可见的）。
- 悬而未决的核心问题（**保持开放**）：正背两侧的"人"是**相同的复制人生**，还是**完全不同的人**？讨论中浮现三种可能：完全镜像 / 完全分立 / 互补纠缠（一个完整灵魂被切成两半，各持互补属性）。**不要替项目拍板。**

### 变体三：双 Σ · 左右对称世界
不是一个 Σ + 介质，而是**两个 Σ 同时诞生**（位于 M₁ 两侧，左右对称）。各自穿过自己的 F，各自展开自己的红/绿/蓝世界，但**共享 M₁ 作为各自世界的第一面墙**。
- 两个世界"对等但非相同"——对称平权，但起点不同、各自独立。
- M₁ 是两个独立宇宙的共同边界（brane 式）。
- 两个 Σ（两个造物主）互相不知道对方存在——各自以为自己是唯一的。只有装置外的观众知道有两个。

### 变体四：旋转对称 · N 世界生成
整个单世界系统绕中心轴旋转复制：
- 旋转 360°/N → 生成 N 个世界。
- N=1 → 原型；N=2（旋转180°，C₂）→ 双世界；N=3（旋转120°，C₃）→ 三世界；N→∞ → 万物。
- 离散旋转群 C_N，连续极限是 SO(2)。对应「一生二、二生三、三生万物」——这是数学事实而非比喻：每次"生"是旋转对称群从低阶到高阶的跃迁。

### 变体五：复合生成序列（最完整形式）
把变体二（正背分裂）+ 变体四（旋转增殖）串联为一个生成序列：
```
Σ+F → 单一平面 → 正反双平面 → 旋转45°混沌交织态 → 正反各自展开红绿蓝 → 旋转复制成多世界
```
- 两种本体论事件：**分裂**（一→二，非对称，阴阳）+ **增殖**（二→N，对称，旋转）。两者不可交换（先分裂再增殖 ≠ 反之）。
- 中心出现一个 **X 空间**（三个 Σ 的第一面镜面之间形成的空间）——**尚无定义**。讨论中浮现的可能定位：未显化母体 / 诸世界之间的中阴 / 神性的背面（造物主们彼此靠近却背对、互不可见的盲区）。**保持开放。**
- 关键认识：前四个变体是变体五的不同截面/投影；生成**顺序可重组**（这是一个"生成语法"而非固定"剧本"）。

## 四、世界的「展开」与「收缩」

装置不是静态几何，而是**由参数（slider）驱动的动力学系统**。每个变体都有动画。

- **展开（unfolding）**：连续流动的过程。从 Σ+F 逐层生成世界。对应流溢 / proodos / "道生一,一生二……"。
- **收缩（folding back）**：设计为一个**降维过程**，不是展开的简单倒放：
  ```
  完整多世界 → 蓝层先撤 → 绿层 → 红层 → 各层面退化成线 → 三条线收成点
  → 回到只有 F 没有 Σ 的中间态 → Σ 重新出现在 F 旁 → 只剩 Σ+F（前创世素材）
  ```
  - 面(2D) → 线(1D) → 点(0D)，逐步降维。
  - "三条线"那一帧是 C₃ 对称的最纯净拓扑骨架。
  - 最终态是 Σ 与 F **并置但尚未进入创世关系**（前创世）。
- **收缩顺序未定（开放）**：取决于如何叙事，目前只列可能性。不同顺序对应不同哲学（五蕴皆空 / 唯识 / 热寂 / 量子退相干 / 层间吞噬）。**不要写死叙事顺序。**

> 观察（供参考，非定论）：展开是连续的、收缩是阶梯式离散的——这种不对称若有意，则暗示这个宇宙有时间方向（非永恒轮回）；也可能只是关键帧选择所致。

## 五、开放方向（请勿强行"完成"）

以下为尚未定论的开放问题。可探讨、提供可能性，但**不要写死**：

1. **反射回路**：最外层镜面 M₃（极大）的反射打回内层镜面 M₁（较小）会怎样？
   - 平面+垂直配置下：大镜面反射回小镜面只能传回**一部分内容，其余超出尺寸而丢失**（朴素几何事实，不要往"曲率/全息"方向过度延伸，除非明确要求）。
   - 真正关心的是：**回流的光会不会反过来影响那一层世界里"人"的状态，从而影响整个反射循环链路？**（装置前提是"光/意识对世界的投射"——回流即反馈）。
   - 不同镜面性质（单向 / 双向 / 半透明 / 双向半透明）会产生不同的反馈/叠加效果。这是从"几何装置"到"带反馈的动力学系统"的潜在升级方向。

2. **"生成新世界"的机制**："两个世界的人同时触碰镜面生成第三世界"这类**事件驱动**设定已弃用（太套路、太人本主义、太生硬）。方向是**符合客观规律、让人觉得理所应当**的机制（候选：临界相变 / 对称破缺 / 驻波节点 / 不动点 / 共形边界条件等"规律驱动而非事件驱动"的方式）。**此问题悬置，不急于解决。**

3. **曲率 / 极小曲面**：可把平面镜换成曲面。曲面会把决定论变为概率性（焦点=命运吸引子，焦散=自然明暗）。极小曲面（零平均曲率、由边界条件涌现）对应"形状不被设计、由约束自然涌现"——与"减少设计者意志强加"的倾向一致。但曲面的计算/模拟成本高，目前用平面（好计算、好建模、好模拟）。

4. **四维时空**：若装置进入 4D，则所有变体是同一个 4D 母体的不同 3D 切片；展开/收缩是第四维上的不同投影方向；bug 区域可能是 4D 旋转的入口。

5. **结构性穿越（参数驱动的换层）**：调整镜面间距或取景框（前后距离 / 大小），让分层边界扫过一个**物理上静止**的人——人不动，世界的划分移动，他便从第一层"瞬移"到第二层。从 Σ 看会经历三个阶段：单一可见（第一层）→ 进入相邻锥重叠区（红区）时**画面中同时出现两个他**（重影态）→ 仅经两次反射可见（第二层，表观尺寸 / 深度改变，且左右手性翻转）。
   - 三种可能结局，各有几何条件与叙事含义：成功换层 / 被甩出光锥（坠入黑暗）/ 滞留重影区。
   - 与解读 A 相连：**层级位置的改变可以不来自人的移动，而来自世界划分标准的移动**；红区由"穿帮区"变为换层的必经通道——重影是穿越的过程态，不是 bug。
   - 该机制是参数驱动、几何必然的（人不做任何事），符合本项目"规律驱动而非事件驱动"的取向。
   - web 版已有最小演示：开启「人物固定世界坐标」后拖动镜距即可观察。何时把它发展成可编排的叙事段落，**保持开放**。

> 贯穿这些方向的深层倾向：**逐步减少"设计者主动定义每个细节"**，转向**让几何/物理规律自己决定形状**（极小曲面、对称破缺、约束求解）。这可能在悄悄把 Σ 从"造物主"降格为"边界条件"。方向如此，尚未定论。

## 六、技术历程

| 时期 | 工具 | 当时在解决的问题 |
|------|------|----------------|
| 2010s 初 | Lumion | 镜面在不同环境/角度下反射什么（兴趣的起点，比装置更早） |
| 2023 | Vectorworks + Twinmotion 同步预览 | 真实物理尺度验证：人体比例、地面位置/高度/范围、潜在 bug。装置是**建筑级尺度**（人在墙下显得很小）。SketchUp 视图里红色线条是真实反射光路轨迹。 |
| 近期 | Rhino + Grasshopper（文件 3-9.gh, GH 8.5） | 几何深层对称性的参数化推演。Mirror×42 + Rotate×85 + Pi×881，纯几何变换，无第三方求解器插件。 |
| 当前 | Twinmotion 路径追踪 + Three.js web 版 | 验证光路是否跑通；做成可交互 web 装置。 |

**Three.js web 版当前状态（持续更新）**：
- 已完成变体一/三的可交互单页。
- **实现路线 = 视锥追迹（frustum ray tracing），不是简化反射矩阵**：从 Σ 发出穿过 F 四角的四条光线，每条独立追迹反射；**镜面 = 视锥在该位置的截面**（镜面尺寸由光锥自然决定，不预设）；层级放大是视锥发散的自然结果。
- 几何已验证正确：45°→光路90°✓、三层递增放大✓、穿帮检测✓（角光线越过镜面边界触发 bug 警告）。
- 用 .gh 真实尺度（原始单位，内部 ÷100 仅为渲染舒适）。
- 已实现"可布景体差集计算"（自动算每层扣除遮挡后的有效布景体积，即 2023 年在 SketchUp 里手动做的事）。
- 架构已从单文件转向系统化模块（CORE 纯数学 / RENDER / APP），含 Node 几何回归测试（见《系统架构与工作计划》）。
- 待办：真实降维收缩动画（面→线→点）、多世界生成过渡、真实镜面反射（Reflector / 路径追踪）。

**工具迭代的轨迹 = 装置内部本体论的展开轨迹**。每次换工具是为了能问更深的问题，不只是技术升级——是工具在追赶想法，不是想法适配工具。

未来愿景：这个 web 模块可能集成进 **Blackplant**（contract-first 时空数据平台，entity/sample 抽象 + 身份置信度 + timeline 机制）。镜面装置与 Blackplant 共享同一底层认识论：**事物（人/植物）不被直接给予，只能通过中介（镜面反射光路 / sample 累积）被间接通达**。两者都是"过程/轨迹"本体论，拒绝"离散给定的实体"。

## 七、协作原则

- **偏好开放探索，容纳歧义，反对过早收口**。许多问题刻意保持开放，强行给"完美答案"不受欢迎。
- **反对过度工程化、过早进入实现细节、过度的哲学脚手架**。"列五个方案 A/B/C/D/E"式的轰炸应被收回到"我们在畅想"。
- **几何精确性优先**：反射方向/比例的错误需被快速、精确地纠正，并**真正吸收**（不是表面承认）。
- **舒适于高度抽象框架**（道家宇宙论、时空几何、本体论认识论），将其当作真正的结构工具而非装饰。
- 中英文皆可；技术沟通可用专业术语，不必过度解释基础概念。
- **长时间尺度**：这是 15+ 年的项目，追求"持续生长"而非"完成"，不设完成日期。
